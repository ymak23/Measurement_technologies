Лабораторная работа № 3

Добротность ПКЭ определяется по значениям квадрата модуля проводимости ПКЭ в области резонанса.

Цель работы: исследование влияния добротности ПКЭ на точность определения самой добротности по значениям квадрата модуля проводимости ПКЭ в области резонанса.

Описание программы «Лабораторная работа 3»


Запустить программу «Лабораторная работа 3» любым известным способом. После запуска программы появляется диалоговое окно работы с программой, показанное на Рисунке 1.
В окне программы выделены три области:
1 – область задания параметров элементов эквивалентной электрической схемы (для ввода данных необходимо установить курсор в ячейку и набрать нужное число),
2 – область отображения результатов расчета,
3 – область отображения графика частотной зависимости модуля проводимости.
